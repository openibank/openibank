Architecture

Designed for Scale

A layered architecture that separates concerns while maintaining tight integration. Click each layer to explore the components.

Surface Layer API
REST API GraphQL WebSocket MCP Server CLI

The Surface Layer provides multiple interface options for interacting with OpeniBank. REST and GraphQL APIs for traditional integrations, WebSocket for real-time updates, MCP (Model Context Protocol) for AI agent communication, and a powerful CLI for developers.

Agent Layer AI
Agent Kernel LLM Bridge Guard Receipts Maple Runtime

The Agent Layer manages AI agent lifecycles and their interactions with the financial system. The Guard component enforces commitment boundaries, ensuring fail-closed safety. Maple Runtime provides the execution environment for agent-defined behaviors.

Financial Layer Core
Ledger Wallet Permits Escrow Clearing Settlement

The Financial Layer implements core banking primitives. Double-entry ledger for accounting, wallet management for asset custody, permit system for authorization, and escrow services for conditional transfers. Clearing and settlement ensure atomic transaction finality.

ResonanceX Exchange Trading
Orderbook Matching Engine Market Data Arena Trading Bots

ResonanceX is our high-performance exchange infrastructure. The matching engine achieves sub-microsecond latency for order matching. Arena provides sandboxed environments for bot testing, while the Trading Bots framework enables automated strategy deployment.

examples/create_agent.rs 
use openibank_sdk::{Client, AgentConfig, Amount};

async fn main() -> Result<()> {
    // Initialize the OpeniBank client
    let client = Client::new("your-api-key").await?;

    // Create a new AI agent with spending limits
    let agent = client
        .create_agent(AgentConfig {
            name: "trading-bot-alpha".into(),
            daily_limit: Amount::usd(10_000),
            capabilities: vec!["trade", "transfer"],
        })
        .await?;

    // Fund the agent's wallet
    client
        .transfer(&agent.wallet_id, Amount::usd(1_000))
        .await?;

    println!("Agent created: {}", agent.id);
    Ok(())
}

import { Client, AgentConfig, Amount } from '@openibank/sdk';

async function main() {
    // Initialize the OpeniBank client
    const client = await Client.connect('your-api-key');

    // Create a new AI agent with spending limits
    const agent = await client.createAgent({
        name: 'trading-bot-alpha',
        dailyLimit: Amount.usd(10_000),
        capabilities: ['trade', 'transfer'],
    } as AgentConfig);

    // Fund the agent's wallet
    await client.transfer(agent.walletId, Amount.usd(1_000));

    console.log(`Agent created: ${agent.id}`);
}

main();
